Ordered list

It is used to display a list in specific order

ol tag used to create an ordered list

The default order is number(1,2,3,...)

type attribute helps to change the default order(type="A","a","i","I","1")

reversed attribute helps to change the reverse the order

start attribute helps to starts from the specific range
